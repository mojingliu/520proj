Week 1

Introductory exercises

  1. Create a homepage if you do not already have one. Where is it? How and why might you use Java applets and other interactive web services on your site? Why might you not want to use them?

  2. Compare the C and Java programming languages. How does choice of language influence programming style? Is object-oriented programming possible in C? Why or why not?

  3. What UNIX tools do you know about that are useful for developing C programs? What about tools for developing Java programs?

  4. Which one or two other class members would you like in your project group?
